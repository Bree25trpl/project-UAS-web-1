# Proses Pengembangan Website

Website ini dikembangkan melalui beberapa tahapan agar menghasilkan tampilan yang menarik, mudah digunakan, dan sesuai dengan tujuan proyek.

1. **Studi Referensi dan Inspirasi**
   Sebelum memulai pengembangan, kami mencari referensi desain dari beberapa website.
   Salah satu inspirasi utama yang digunakan adalah **Passion Made Possible**.
   Inspirasi tersebut menjadi acuan dalam menentukan konsep tata letak (layout),
   serta penyusunan elemen antarmuka, tanpa menyalin isi maupun keseluruhan desainnya secara langsung.

2. **Perancangan Antarmuka (UI) di Figma**
   Setelah menentukan konsep, kami membuat rancangan tampilan website menggunakan **Figma**.
   Tahap ini bertujuan untuk memvisualisasikan struktur halaman, penempatan komponen,
   serta alur navigasi sebelum proses implementasi dimulai.

3. **Implementasi Menggunakan HTML dan CSS**
   Berdasarkan desain yang telah dibuat di Figma, website kemudian diimplementasikan menggunakan **HTML**
   untuk membangun struktur halaman dan **CSS** untuk mengatur tata letak, warna, tipografi, animasi,
   serta tampilan responsif sehingga website lebih menarik dan nyaman digunakan.

4. **Implementasi JavaScript**
   JavaScript digunakan untuk menambahkan interaktivitas pada website, seperti fitur slider, navigasi,
   dan fungsi lainnya. Dalam proses pengembangan, kami juga memanfaatkan **AI (ChatGPT)** sebagai asisten
   pembelajaran untuk membantu mengidentifikasi kesalahan logika, memberikan penjelasan terhadap error
   yang muncul, serta memberikan saran perbaikan kode. Seluruh keputusan implementasi, pengujian, dan
   integrasi tetap dilakukan oleh tim pengembang.

5. **Uji Coba dan Perbaikan**
   Setelah seluruh fitur selesai dikembangkan, website diuji pada berbagai ukuran layar dan skenario penggunaan
   untuk memastikan seluruh fungsi berjalan dengan baik. Setiap bug atau kendala yang ditemukan selama pengujian
   diperbaiki secara bertahap, baik pada tampilan (UI), responsivitas, maupun fungsionalitas JavaScript,
   sehingga website menjadi lebih stabil dan nyaman digunakan.

6. **Validasi Responsivitas**
   Untuk memastikan website dapat digunakan dengan baik pada berbagai perangkat, dilakukan pengujian responsivitas
   menggunakan Chrome DevTools (Device Toolbar). Pengujian dilakukan dengan mensimulasikan beberapa ukuran layar perangkat,
   seperti desktop, tablet, dan smartphone.
